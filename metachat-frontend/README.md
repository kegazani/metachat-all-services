# MetaChat Desktop Frontend

Vue.js десктопное приложение для MetaChat - платформы для ведения личного дневника с AI-анализом настроения и интеллектуальным подбором собеседников.

## Технологии

- **Vue 3** - прогрессивный JavaScript фреймворк
- **Vite** - быстрый сборщик и dev-сервер
- **Vue Router** - маршрутизация
- **Pinia** - управление состоянием
- **Element Plus** - UI компоненты
- **Axios** - HTTP клиент

## Функциональность

### Авторизация
- Регистрация нового пользователя
- Вход в систему
- Автоматическая проверка токена

### Дневник
- Создание, редактирование и удаление записей
- Просмотр записей с анализом настроения
- Теги для организации записей
- Сессии дневника

### Профиль
- Редактирование профиля пользователя
- Просмотр статистики
- Отображение личности (Big Five)
- Прогресс заполнения профиля

### Матчинг
- Просмотр запросов на общение
- Принятие/отклонение запросов
- Фильтрация по статусу
- Отображение совпадений и общих тем

### Чат
- Список активных чатов
- Отправка и получение сообщений
- История сообщений
- Отметка прочитанных сообщений

### Аналитика
- Эмоциональный профиль
- Популярные темы
- Статистика записей

## Установка и запуск

### Требования
- Node.js 18+ 
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:3000`

### Сборка для продакшена

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

### Предпросмотр продакшен сборки

```bash
npm run preview
```

## Конфигурация

### API Endpoint

По умолчанию приложение использует прокси для API запросов. Настройка в `vite.config.js`:

```javascript
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:8080',
      changeOrigin: true,
      rewrite: (path) => path.replace(/^\/api/, '')
    }
  }
}
```

Если API Gateway работает на другом порту, измените `target` в конфигурации.

## Использование

1. Запустите backend сервисы (API Gateway на порту 8080)
2. Запустите frontend: `npm run dev`
3. Откройте браузер: `http://localhost:3000`
4. Зарегистрируйтесь или войдите в систему
5. Начните использовать приложение!

## Особенности

- **Адаптивный дизайн** - работает на разных размерах экрана
- **Автоматическая авторизация** - токен сохраняется в localStorage
- **Обработка ошибок** - автоматический редирект на логин при 401
- **Реактивность** - обновление данных в реальном времени
- **Русская локализация** - все интерфейсы на русском языке

## Разработка

### Добавление нового API endpoint

1. Добавьте метод в соответствующий файл в `src/api/`
2. Используйте его в store или компоненте

### Добавление новой страницы

1. Создайте компонент в `src/views/`
2. Добавьте маршрут в `src/router/index.js`
3. Добавьте пункт меню в `src/layouts/MainLayout.vue` (если нужно)
